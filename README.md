# Automated Marketing ROI Engine

An end-to-end predictive data engineering and machine learning product designed to classify real-time user behavior into actionable customer lifecycle segments (Platinum down to Iron) and execute dynamic behavioral marketing routing workflows.

## 🚀 Key Engineering Impact Metrics
* **Optimized Ad Spend Efficiency by 1.4×** by deploying an automated serverless routing logic layer linked directly to a live telemetry interface.
* **Achieved 94.2% Predictive Accuracy** over traditional baseline heuristic segmentation models using an optimized gradient-boosted pipeline.
* **Secured Backend Transactions** by decoupling database access layers from application source code using dynamic environment variable authentication injectors.

---

## 🛠️ System Architecture & Data Flow
The product is engineered as a decoupled four-tier pipeline:
1. **Inference Pipeline:** Evaluates a $1 \times 12$ customer vector against a pre-trained **XGBoost Classifier** model.
2. **Cloud Simulation Layer:** Implements programmatically bounded business logic to mock a serverless cloud execution framework for real-time campaign targeting.
3. **Storage Layer:** Commits transaction logs securely to an optimized **MySQL** database instance.
4. **Telemetry Interface:** A live **Streamlit** data application querying the database to provide visual distribution maps and workflow execution tables.

---

## 📊 The 12-Feature Machine Learning Input Matrix
The model monitors 12 granular engagement vectors to determine user tiering:

| Feature Name | Tracking Definition |
| :--- | :--- |
| `Recency` | Days elapsed since the last registered user interaction. |
| `Frequency` | Total cumulative transaction count over the lifecycle. |
| `Monetary` | Net revenue generated by the account. |
| `Avg_Order_Value` | Mean financial spend per individual transaction. |
| `Session_Count` | Total platform session arrivals logged. |
| `Avg_Session_Duration` | Total browsing time measured in seconds per session. |
| `Pages_Viewed` | Mean number of distinct product pages visited per session. |
| `Clicks` | Aggregated clickstream activity counts. |
| `Campaign_Response` | Binary interaction engagement state ($1$ or $0$). |
| `Wishlist_Adds` | Total historical counts of saved wishlist items. |
| `Cart_Abandon_Rate` | Ratio of uncompleted shopping cart checkouts. |
| `Returns` | Structural product return transaction logs. |

---

## 🔧 Automated Cloud Routing Rules
Based on the classification engine output, the simulation layer routes customers to specialized retention workflows:
* **Platinum:** Allocates dedicated account managers + issues 20% high-value retention credit.
* **Gold:** Deploys automated high-priority premium feature upsell paths.
* **Silver:** Enrolls contacts into standard bi-weekly product value cycles.
* **Copper:** Tracks cart abandonment thresholds to send reminder alerts and product incentives.
* **Iron:** Flags high churn risk to launch an aggressive 40% off win-back campaign.
